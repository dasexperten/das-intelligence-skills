# Content Repurposer

## Purpose

Turn one strong content asset into useful variations for social, email, blog, marketplace, sales, video, and website channels.

## When to Use

Use this skill when the task is to repurpose articles, videos, research, reviews, FAQs, product pages, webinars, or campaign content.

## Inputs

Useful inputs: source content, target channels, audience, goal, tone, key points, proof points, and CTA.

## Workflow

1. Extract core idea, proof, and useful fragments.
2. Match fragments to channel formats.
3. Rewrite for each channel's behavior and length.
4. Keep message consistent while adapting format.
5. Add CTA and publishing notes.

## Output

Always provide source summary, repurposing map, channel versions, CTA notes, and next actions.

## Common Mistakes

- Copying the same text everywhere.
- Losing the main message.
- Ignoring channel behavior.
- No CTA.
