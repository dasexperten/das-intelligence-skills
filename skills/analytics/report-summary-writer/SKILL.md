# Report Summary Writer

## Purpose

Turn reports, tables, dashboards, and metric exports into clear business summaries with decisions and next actions.

## When to Use

Use this skill when summarizing weekly reports, monthly dashboards, campaign reports, marketplace reports, website reports, or operations reports.

## Inputs

Useful inputs: report data, date range, goal, audience, key numbers, comparison period, issues, wins, risks, and desired output length.

## Workflow

1. Define audience and report goal.
2. Extract key numbers and changes.
3. Separate wins, issues, and risks.
4. Explain what it means.
5. Recommend next actions.

## Output

Always provide executive summary, key numbers, wins, issues, risks, decisions, and action list.

## Common Mistakes

- Rewriting every number.
- No meaning.
- No decision link.
- No action list.
