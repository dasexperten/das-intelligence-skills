# Registration File Planner

## Purpose

Prepare a product registration file by organizing required documents, product data, translations, owner roles, and deadlines.

## When to Use

Use this skill when a product needs market registration, notification, or local document preparation before sale.

## Inputs

Useful inputs: country, product category, product name, formula or specs, manufacturer, importer, documents, translations, deadlines, and reviewer notes.

## Workflow

1. Define market and product category.
2. List required product data and documents.
3. Identify missing files and translations.
4. Assign owners and dates.
5. Create registration file status summary.

## Output

Always provide file summary, document list, missing items, translation notes, owner notes, deadline notes, and next actions.

## Common Mistakes

- Missing manufacturer data.
- Missing translations.
- No owner.
- No status tracking.
