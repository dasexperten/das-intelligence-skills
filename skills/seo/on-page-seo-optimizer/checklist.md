# On-Page SEO Optimizer Checklist

- [ ] Target query group is clear.
- [ ] Search intent is defined.
- [ ] Title and meta are reviewed.
- [ ] H1 and headings are logical.
- [ ] Content answers the query.
- [ ] Missing sections are identified.
- [ ] FAQ ideas are included.
- [ ] Internal links are planned.
- [ ] Conversion path is considered.
