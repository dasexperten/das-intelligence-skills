# Programmatic SEO Planner Checklist

- [ ] Page pattern has real search demand.
- [ ] Each page has unique value.
- [ ] Data source is reliable.
- [ ] Required fields are mapped.
- [ ] Template modules are defined.
- [ ] Internal links are planned.
- [ ] Canonical and indexation rules are clear.
- [ ] Quality thresholds are defined.
- [ ] Test set is planned before scaling.
