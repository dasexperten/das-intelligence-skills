# Site Architecture Planner Checklist

- [ ] Business goal is clear.
- [ ] Priority page types are defined.
- [ ] Category hierarchy is logical.
- [ ] URL logic is clear.
- [ ] Navigation supports key pages.
- [ ] Breadcrumbs are considered.
- [ ] Internal links are planned.
- [ ] Crawl depth is reviewed.
- [ ] Market and language structure is considered.
- [ ] Risks and next actions are included.
