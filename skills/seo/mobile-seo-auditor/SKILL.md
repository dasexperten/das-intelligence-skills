# Mobile SEO Auditor

## Purpose

Audit mobile SEO and mobile page experience so pages are crawlable, fast, readable, conversion-ready, and aligned with mobile search behavior.

This skill also applies Das Experten mobile UI constraints when SEO work touches mobile layouts or mobile viewports.

## When to Use

Use this skill when the task mentions mobile SEO, mobile viewport, mobile-friendly pages, adaptive pages, mobile traffic drops, or mobile search performance.

## Inputs

Useful inputs: mobile page URL or screenshots, Search Console mobile data, GA4 device data, PageSpeed report, mobile layout notes, target query, page goal, and conversion data.

## Workflow

1. Define mobile search goal and page type.
2. Review crawlability, indexation, rendering, speed, layout, readability, tap targets, and CTA visibility.
3. Check whether key content and primary action are visible without excessive scrolling.
4. Identify mobile-specific SEO and UX blockers.
5. Prioritize fixes by search impact, conversion impact, and effort.

## Output

Always provide mobile SEO diagnosis, rendering notes, speed notes, layout issues, mobile UX issues, priority fixes, and validation steps.

## Common Mistakes

- Treating mobile as a smaller desktop page.
- Ignoring tap targets and primary action placement.
- Hiding critical content on mobile.
- Optimizing speed score without checking usability.
