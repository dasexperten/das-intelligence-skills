# Mobile SEO Auditor Checklist

- [ ] Mobile search goal is clear.
- [ ] Mobile rendering is checked.
- [ ] Page speed is reviewed.
- [ ] Key content is visible.
- [ ] Tap targets are usable.
- [ ] Primary action is reachable.
- [ ] Mobile layout avoids desktop-only patterns.
- [ ] Search Console mobile data is reviewed.
- [ ] Priority fixes are practical.
- [ ] Validation steps are included.
