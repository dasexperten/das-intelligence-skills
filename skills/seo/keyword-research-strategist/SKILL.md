# Keyword Research Strategist

## Purpose

Build practical keyword and search-intent maps that guide SEO content, category pages, product pages, landing pages, and international expansion.

## When to Use

Use this skill when the task is to research keywords, group search intent, choose priority queries, create topic clusters, or map keywords to pages.

## Inputs

Useful inputs: market, language, product category, seed keywords, competitors, Search Console data, target pages, buyer journey, and business priorities.

## Workflow

1. Define market, language, and business goal.
2. Group queries by intent: informational, commercial, transactional, navigational, local, or branded.
3. Remove irrelevant or low-intent terms.
4. Map priority query groups to page types.
5. Identify content gaps and quick wins.
6. Prioritize by relevance, difficulty, value, and readiness.

## Output

Always provide keyword groups, search-intent map, page mapping, content gaps, priority list, and next actions.

## Common Mistakes

- Chasing volume without intent.
- Mixing languages or markets.
- Assigning the same query to many pages.
- Ignoring existing pages.
