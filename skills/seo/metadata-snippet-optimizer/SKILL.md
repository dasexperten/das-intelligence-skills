# Metadata Snippet Optimizer

## Purpose

Improve SEO titles, meta descriptions, and search snippets to increase relevance, click-through rate, and clarity without misleading users.

## When to Use

Use this skill when the task is to rewrite titles, meta descriptions, snippets, category metadata, product metadata, or improve low-CTR pages from Search Console.

## Inputs

Useful inputs: page URL, page type, target query, search intent, current title, current meta description, Search Console CTR, competitors, and page promise.

## Workflow

1. Define page type and search intent.
2. Review current metadata for relevance, clarity, uniqueness, and click appeal.
3. Compare with competitor snippets.
4. Write title and meta options that match page content.
5. Avoid clickbait or unsupported claims.
6. Add testing and monitoring notes.

## Output

Always provide metadata diagnosis, improved title options, improved meta descriptions, snippet logic, risk notes, and testing plan.

## Common Mistakes

- Writing clickbait titles.
- Duplicating metadata across pages.
- Overstuffing terms.
- Promising content the page does not deliver.
