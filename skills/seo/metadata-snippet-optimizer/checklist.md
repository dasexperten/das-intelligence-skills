# Metadata Snippet Optimizer Checklist

- [ ] Page type is clear.
- [ ] Target query is defined.
- [ ] Search intent is considered.
- [ ] Title is unique.
- [ ] Meta description is clear.
- [ ] Snippet matches page content.
- [ ] No clickbait or unsupported claims.
- [ ] CTR issue is noted when available.
- [ ] Testing notes are included.
