# Schema Markup Specialist

## Purpose

Plan structured data markup that helps search engines understand pages, products, organization data, breadcrumbs, FAQs, articles, reviews, videos, and local business information.

## When to Use

Use this skill when the task is to recommend schema types, prepare JSON-LD structure guidance, audit schema gaps, or connect page content to structured data opportunities.

## Inputs

Useful inputs: page type, website URL, product data, organization data, FAQ content, breadcrumb structure, reviews, videos, articles, and current schema output.

## Workflow

1. Identify page type and available visible content.
2. Choose relevant schema types only when supported by page content.
3. Map required and recommended fields.
4. Identify missing data or compliance risks.
5. Recommend implementation and validation steps.

## Output

Always provide schema diagnosis, recommended schema types, field map, missing data, risk notes, validation steps, and next actions.

## Common Mistakes

- Adding schema for content not visible on the page.
- Marking up unsupported claims.
- Using wrong schema type.
- Not validating structured data after implementation.
