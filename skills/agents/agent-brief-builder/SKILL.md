# Agent Brief Builder

## Purpose

Create clear task briefs for AI agents so the agent knows the goal, inputs, rules, workflow, output format, and limits.

## When to Use

Use this skill before giving an agent a complex task, repeatable workflow, or multi-step business job.

## Inputs

Useful inputs: task goal, context, source files, rules, required output, examples, constraints, and success criteria.

## Workflow

1. Define goal and user intent.
2. List inputs and sources.
3. Write task rules and limits.
4. Define workflow steps.
5. Define output format and success criteria.

## Output

Always provide agent brief, inputs list, workflow, output format, checks, and next actions.

## Common Mistakes

- Vague goal.
- Missing input list.
- No output format.
- No quality check.
