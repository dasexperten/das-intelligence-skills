# Agent Handoff Manager

## Purpose

Prepare clean handoffs between agents, people, tools, or work sessions.

## When to Use

Use this skill when work must continue later, move to another person, or transfer from one workflow to another.

## Inputs

Useful inputs: task, current status, completed work, open items, files, owners, decisions, risks, and next action.

## Workflow

1. Summarize current status.
2. List completed work.
3. List open items and decisions.
4. Identify files and owners.
5. Define next action.

## Output

Always provide handoff summary, completed work, open items, file list, owner notes, and next action.

## Common Mistakes

- No status summary.
- Missing files.
- No owner.
- No next action.
