# Agents and Director Skills

This folder contains skills for agent briefs, routing, output review, handoff, and director-level management briefs.

## Completed Skills

1. `agent-brief-builder`
2. `agent-router`
3. `agent-output-reviewer`
4. `agent-handoff-manager`
5. `director-daily-brief`
6. `director-decision-brief`
7. `director-execution-tracker`
8. `director-dashboard-reviewer`

## Usage Logic

- new agent task → `agent-brief-builder`
- choose workflow → `agent-router`
- review result → `agent-output-reviewer`
- continue work → `agent-handoff-manager`
- daily management → `director-daily-brief`
- business decision → `director-decision-brief`
- project follow-up → `director-execution-tracker`
- dashboard review → `director-dashboard-reviewer`

Agents and Director foundation is complete for the current KISS stage.
