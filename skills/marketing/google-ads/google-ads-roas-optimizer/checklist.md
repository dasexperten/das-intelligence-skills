# Google Ads ROAS Optimizer Checklist

- [ ] Conversion values are reliable.
- [ ] Margin and break-even ROAS are known.
- [ ] Actual and target ROAS are compared.
- [ ] Conversion delay is excluded from evaluation.
- [ ] Products and markets are segmented.
- [ ] Low-margin value is identified.
- [ ] Target changes are gradual.
- [ ] Scale and stop rules are defined.
