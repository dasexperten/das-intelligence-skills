# Offer Creation Expert

## Purpose

Create stronger commercial offers that make a product, service, bundle, launch, or B2B proposal easier to understand, compare, trust, and buy.

This skill improves the offer itself, not just the wording around it.

## When to Use

Use this skill when the task is to:

- create an offer;
- improve conversion;
- build a bundle;
- prepare a launch offer;
- improve marketplace promotions;
- create a B2B proposal angle;
- reduce price resistance;
- increase perceived value.

## When NOT to Use

Do not use when the task is only brand positioning, legal contract wording, detailed unit economics, or ad creative. Use this after the core product and target customer are known.

## Inputs

Useful inputs:

- product or service;
- target customer;
- channel;
- price;
- margin limits;
- competitors;
- customer objections;
- bonuses or bundles;
- urgency reason;
- desired action.

## Workflow

1. Define the customer and desired action.
2. Identify the current friction: price, trust, effort, risk, timing, or confusion.
3. Clarify the core promise and main outcome.
4. Add value elements only if they increase buying confidence.
5. Reduce risk with proof, guarantee, clarity, or comparison.
6. Add urgency only when there is a real reason.
7. Package the offer into a simple structure.
8. Create channel-specific versions for website, marketplace, ads, and B2B.
9. Recommend tests for price, bundle, headline, proof, and call to action.

## Output

Always provide:

- offer diagnosis;
- target customer;
- core promise;
- value stack;
- risk reducers;
- urgency or timing logic;
- final offer structure;
- channel versions;
- test ideas.

## Best Practices

- Improve the deal before improving the copy.
- Make the value easy to compare.
- Reduce perceived risk before pushing urgency.
- Use real bonuses, not filler.
- Keep the offer simple enough to explain in one screen.
- Match the offer to channel behavior.

## Common Mistakes

- Discounting too early.
- Adding bonuses nobody wants.
- Using fake urgency.
- Making the offer too complex.
- Ignoring margin and operational limits.
- Confusing a slogan with an offer.
