# Offer Creation Expert Checklist

- [ ] Target customer is clear.
- [ ] Desired action is clear.
- [ ] Main friction is identified.
- [ ] Core promise is specific.
- [ ] Value stack contains useful elements.
- [ ] Risk reducers are included.
- [ ] Proof supports the offer.
- [ ] Urgency has a real reason.
- [ ] Offer is simple to explain.
- [ ] Test ideas are practical.
