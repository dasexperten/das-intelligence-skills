# Storytelling Architect Checklist

- [ ] Audience is specific.
- [ ] Story goal is clear.
- [ ] Desired belief shift is defined.
- [ ] Core conflict or problem is identified.
- [ ] Transformation is clear.
- [ ] Story type is selected.
- [ ] Proof points are included.
- [ ] Story is adapted to channel.
- [ ] Short version is usable.
- [ ] Long version outline is structured.
- [ ] CTA or next action is clear.
