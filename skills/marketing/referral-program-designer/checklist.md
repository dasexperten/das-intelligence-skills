# Referral Program Designer Checklist

- [ ] Referral goal is clear.
- [ ] Target referrer is specific.
- [ ] Referred customer is defined.
- [ ] Referral motivation is understood.
- [ ] Incentive fits margin and behavior.
- [ ] Referral action is simple.
- [ ] Program rules are clear.
- [ ] Tracking logic is included.
- [ ] Fraud and abuse risks are considered.
- [ ] Launch plan is practical.
- [ ] KPIs measure both volume and quality.
