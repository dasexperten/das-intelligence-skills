# Customer Journey Designer

## Purpose

Map and improve the customer journey from first awareness to purchase, repeat purchase, referral, or B2B partnership.

This skill helps identify friction, missing proof, weak touchpoints, and opportunities to improve conversion and retention across websites, marketplaces, ads, content, social, email, and sales.

## When to Use

Use this skill when the task is to:

- map a customer journey;
- find conversion friction;
- improve customer experience;
- connect touchpoints across channels;
- plan content for each buying stage;
- improve repeat purchase;
- diagnose where customers drop off.

## When NOT to Use

Do not use when the user only needs one landing page, one ad, one marketplace card, or a technical analytics setup. Use this skill before specialist execution when the journey is unclear.

## Inputs

Useful inputs:

- product or brand;
- customer segment;
- channel mix;
- current funnel;
- website or marketplace data;
- reviews and objections;
- sales process;
- email or retargeting flows;
- known drop-off points.

## Workflow

1. Define the target customer and desired final action.
2. Map the stages: awareness, consideration, trust, purchase, onboarding, repeat, referral.
3. Identify customer questions and doubts at each stage.
4. Match touchpoints to each stage.
5. Identify friction, missing proof, and message gaps.
6. Recommend content, offer, UX, and follow-up improvements.
7. Prioritize fixes by impact and effort.
8. Define metrics for each stage.

## Output

Always provide:

- journey map;
- customer questions by stage;
- touchpoints;
- friction points;
- proof gaps;
- improvement actions;
- KPI by stage;
- priority roadmap.

## Best Practices

- Map the real customer path, not the ideal company process.
- Include doubts and risk perception at each stage.
- Connect messaging, UX, proof, and follow-up.
- Focus first on the biggest drop-off or trust gap.
- Keep the journey simple enough to act on.

## Common Mistakes

- Creating a beautiful map with no actions.
- Ignoring post-purchase experience.
- Treating all customers as one journey.
- Missing marketplace-specific touchpoints.
- Measuring only traffic and sales while ignoring trust and retention.
