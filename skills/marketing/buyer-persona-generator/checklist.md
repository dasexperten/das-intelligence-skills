# Buyer Persona Generator Checklist

- [ ] Persona is based on motivation or use case.
- [ ] Buying trigger is clear.
- [ ] Main problem is specific.
- [ ] Desired outcome is practical.
- [ ] Objections are included.
- [ ] Decision criteria are listed.
- [ ] Proof needs are clear.
- [ ] Channel behavior is included.
- [ ] Persona produces useful message angles.
- [ ] Persona is not a fictional biography.
