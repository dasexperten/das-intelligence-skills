## Usage Logic

Use strategy skills first when the business direction is unclear:

- `brand-strategy-expert`
- `brand-positioning-expert`
- `value-proposition-architect`
- `marketing-planner`
- `growth-strategy-expert`

Use execution planning skills when the direction is already clear:

- `product-marketing-expert`
- `go-to-market-strategist`
- `launch-campaign-planner`
- `market-entry-strategist`
- `pricing-strategy-expert`

Use customer and market understanding skills when evidence is needed:

- `customer-research-specialist`
- `buyer-persona-generator`
- `competitive-intelligence-analyst`
- `customer-journey-designer`

Use trust, loyalty, and communication skills when the goal is reputation or long-term relationship building:

- `retention-specialist`
- `referral-program-designer`
- `community-marketing-expert`
- `public-relations-expert`
- `crisis-communications-expert`
- `storytelling-architect`
