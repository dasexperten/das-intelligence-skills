# Public Relations Expert Checklist

- [ ] PR goal is clear.
- [ ] Target audience is defined.
- [ ] Newsworthy angle is specific.
- [ ] Key messages are simple.
- [ ] Proof points support the story.
- [ ] Media target categories are selected.
- [ ] Press materials are listed.
- [ ] Outreach plan is practical.
- [ ] Follow-up rhythm is included.
- [ ] Risks and approvals are considered.
- [ ] KPIs match the PR goal.
