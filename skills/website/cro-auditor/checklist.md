# CRO Auditor Checklist

- [ ] Conversion goal is clear.
- [ ] Traffic source is considered.
- [ ] Target audience is specific.
- [ ] First-screen clarity is reviewed.
- [ ] CTA is visible and relevant.
- [ ] Trust signals are checked.
- [ ] Main objections are identified.
- [ ] Mobile friction is considered.
- [ ] Checkout or form friction is reviewed.
- [ ] Fixes are prioritized by impact and effort.
- [ ] KPIs and decision rules are included.
