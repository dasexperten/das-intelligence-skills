# Conversion Copywriter

## Purpose

Write and improve website copy that helps visitors understand value, believe claims, overcome objections, and take action.

This skill is built for homepages, product pages, landing pages, CTAs, FAQs, banners, forms, and checkout microcopy.

## When to Use

Use this skill when the task is to:

- rewrite website copy;
- improve conversion messaging;
- create headlines and CTAs;
- turn features into benefits;
- address objections;
- improve product or landing page copy;
- make copy clearer, sharper, or more persuasive.

## When NOT to Use

Do not use when the user needs a full website audit, analytics analysis, or brand strategy from scratch. Use this skill for conversion-focused copy execution.

## Inputs

Useful inputs:

- page type;
- target audience;
- product or offer;
- current copy;
- value proposition;
- benefits;
- proof points;
- objections;
- tone;
- desired CTA;
- traffic source.

## Workflow

1. Define the page goal and reader intent.
2. Clarify the core promise.
3. Translate features into benefits.
4. Build headline, subheadline, body, proof, and CTA hierarchy.
5. Address objections with concise copy.
6. Improve clarity, specificity, and flow.
7. Create variants for testing when useful.

## Output

Always provide:

- copy diagnosis;
- improved headline;
- improved subheadline;
- section copy;
- CTA options;
- objection-handling copy;
- proof placement notes;
- test variants.

## Best Practices

- Write for the buyer's decision, not the company's preference.
- Use specific benefits and proof.
- Keep CTAs action-oriented and clear.
- Remove vague adjectives.
- Match copy to the traffic source and buying stage.

## Common Mistakes

- Clever copy that is not clear.
- Claims without proof.
- Too many messages in one headline.
- Generic CTAs.
- Long paragraphs where scanning is needed.
