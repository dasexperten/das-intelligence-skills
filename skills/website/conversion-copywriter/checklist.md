# Conversion Copywriter Checklist

- [ ] Page goal is clear.
- [ ] Target reader is specific.
- [ ] Core promise is visible.
- [ ] Headline is clear before clever.
- [ ] Features are translated into benefits.
- [ ] Proof supports claims.
- [ ] Objections are addressed.
- [ ] CTA is specific.
- [ ] Copy is easy to scan.
- [ ] Test variants are useful.
