# Landing Page Builder

## Purpose

Create focused landing pages that match a specific audience, traffic source, offer, promise, proof, and conversion goal.

This skill is built for campaign pages, product launches, lead capture pages, B2B offers, marketplace-to-site traffic, and paid ads funnels.

## When to Use

Use this skill when the task is to:

- build a landing page;
- structure a campaign page;
- create landing page copy outline;
- match page content to an ad or traffic source;
- improve lead capture or purchase conversion;
- create a one-page funnel.

## When NOT to Use

Do not use for full homepage strategy, deep product page optimization, or checkout flow redesign. Use this skill for focused single-goal pages.

## Inputs

Useful inputs:

- landing page goal;
- target audience;
- traffic source;
- offer;
- product or service;
- proof points;
- objections;
- CTA;
- brand tone;
- deadline or campaign context.

## Workflow

1. Define the conversion goal and visitor intent.
2. Clarify the promise and offer.
3. Create hero section: headline, subheadline, proof, CTA.
4. Build benefit and proof sections.
5. Address objections and risks.
6. Add social proof, FAQs, and urgency when relevant.
7. Create section order and copy blocks.
8. Define measurement and testing plan.

## Output

Always provide:

- landing page strategy;
- hero section;
- section order;
- copy outline;
- proof and trust elements;
- FAQ and objection handling;
- CTA plan;
- KPI and test notes.

## Best Practices

- One page should have one main goal.
- Match the headline to traffic source intent.
- Put proof close to the promise.
- Remove navigation or distractions when appropriate.
- Make the CTA specific and repeated logically.

## Common Mistakes

- Building a generic page for every audience.
- Weak headline that does not match the ad.
- Too many CTAs.
- Claims without proof.
- Long page with no conversion logic.
