# Site Speed Auditor Checklist

- [ ] Page type is clear.
- [ ] Device is defined.
- [ ] Core Web Vitals are reviewed.
- [ ] Image size and format issues are checked.
- [ ] Script and app weight is reviewed.
- [ ] Font loading is considered.
- [ ] Layout shift issues are noted.
- [ ] Mobile performance is prioritized when relevant.
- [ ] Fixes are prioritized by impact and effort.
- [ ] Retest plan is included.
