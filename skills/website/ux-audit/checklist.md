# UX Audit Checklist

- [ ] User task is clear.
- [ ] Navigation is understandable.
- [ ] Page hierarchy is logical.
- [ ] Important actions are visible.
- [ ] Content is easy to scan.
- [ ] Mobile experience is reviewed.
- [ ] Forms and buttons are usable.
- [ ] Labels are clear.
- [ ] Accessibility issues are noted.
- [ ] Fixes are prioritized by severity and effort.
