# Homepage Optimizer

## Purpose

Improve a homepage so it quickly explains what the brand sells, why it matters, who it is for, why visitors should trust it, and what action they should take next.

This skill is built for e-commerce, DTC brands, B2B websites, marketplace-linked brands, international sites, and product-led homepages.

## When to Use

Use this skill when the task is to:

- improve a homepage;
- audit homepage conversion;
- clarify homepage messaging;
- organize homepage sections;
- improve trust and proof;
- create a homepage wireframe;
- reduce bounce rate;
- improve first-screen clarity.

## When NOT to Use

Do not use when the user only needs product page optimization, checkout optimization, deep SEO audit, or analytics tracking setup. Use the relevant specialist skill for those tasks.

## Inputs

Useful inputs:

- website URL or homepage copy;
- brand positioning;
- target customer;
- main product categories;
- value proposition;
- proof points;
- current performance signals;
- traffic source;
- desired CTA;
- markets and languages.

## Workflow

1. Identify the homepage goal and main audience.
2. Check above-the-fold clarity: what, for whom, why trust, why now, what next.
3. Review value proposition and headline.
4. Map homepage sections in logical order.
5. Identify missing proof, trust signals, and objections.
6. Check CTA clarity and hierarchy.
7. Recommend copy, UX, content, and proof improvements.
8. Prioritize fixes by impact and effort.

## Output

Always provide:

- homepage diagnosis;
- first-screen recommendation;
- value proposition improvement;
- section order;
- trust and proof gaps;
- CTA recommendations;
- priority fixes;
- next steps.

## Best Practices

- Make the first screen understandable in seconds.
- Lead with customer value, not company history.
- Show proof close to claims.
- Use one primary CTA and clear secondary actions.
- Make product categories easy to understand.
- Adapt homepage logic for traffic source and market.

## Common Mistakes

- Vague hero headline.
- Too many CTAs.
- Beautiful design with weak clarity.
- Proof placed too late.
- Product categories unclear.
- Homepage optimized for the company, not the visitor.
