# Task Priority Planner

## Purpose

Create a clear order for business tasks based on value, timing, effort, owner, and dependencies.

## When to Use

Use this skill when there are many tasks and the next action is unclear.

## Inputs

Useful inputs: task list, goal, dates, owners, dependencies, effort, and expected value.

## Workflow

1. Clarify goal.
2. Group tasks by area.
3. Score value, effort, and dependency.
4. Identify fast useful tasks.
5. Create action order.

## Output

Always provide task table, first actions, blocked items, owner notes, timing, and next actions.

## Common Mistakes

- No owner.
- No order.
- No dependency check.
