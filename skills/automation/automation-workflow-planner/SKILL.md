# Automation Workflow Planner

## Purpose

Plan simple automation workflows that connect inputs, tools, data, checks, and outputs.

## When to Use

Use this skill when designing a Make, Zapier, API, Sheets, Gmail, Slack, marketplace, or CRM automation.

## Inputs

Useful inputs: business goal, source data, destination, tools, fields, timing, rules, checks, and desired output.

## Workflow

1. Define business goal and output.
2. Map source and destination.
3. List fields and rules.
4. Add checks and fallback steps.
5. Create workflow plan and test notes.

## Output

Always provide workflow map, field map, rules, checks, test plan, and next actions.

## Common Mistakes

- Automating before process is clear.
- Missing field map.
- No test plan.
- No fallback step.
