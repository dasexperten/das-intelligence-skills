# Logistics Skills

This folder contains practical logistics skills for routes, 3PL evaluation, warehouse inbound, stock planning, landed cost, and shipment documents.

## Completed Logistics Skills

1. `route-planner`
2. `third-party-logistics-evaluator`
3. `warehouse-inbound-planner`
4. `stock-planner`
5. `landed-cost-planner`
6. `shipment-document-planner`

## Usage Logic

- route comparison → `route-planner`
- warehouse or fulfillment partner → `third-party-logistics-evaluator`
- warehouse delivery → `warehouse-inbound-planner`
- stock timing → `stock-planner`
- full arrival cost → `landed-cost-planner`
- shipment paperwork → `shipment-document-planner`
