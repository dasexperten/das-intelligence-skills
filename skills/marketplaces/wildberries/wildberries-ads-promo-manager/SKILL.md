# Wildberries Campaign Manager

## Purpose

Plan Wildberries campaigns so budget supports visibility, sales, stock movement, and margin control.

## When to Use

Use this skill when planning WB campaigns, budgets, discounts, launch pushes, or campaign cleanups.

## Inputs

Useful inputs: SKU, category, budget, CPC, conversion, orders, revenue, stock, price, margin, calendar, and card readiness.

## Workflow

1. Define campaign goal.
2. Check card readiness and stock.
3. Review budget, CPC, conversion, and economics.
4. Plan structure and timing.
5. Set KPIs, stop rules, and next actions.

## Output

Always provide diagnosis, SKU priority, budget logic, calendar plan, KPI notes, risks, and next steps.

## Common Mistakes

- Scaling budget before card is ready.
- No stop rule.
- Ignoring stock.
- Ignoring margin.
