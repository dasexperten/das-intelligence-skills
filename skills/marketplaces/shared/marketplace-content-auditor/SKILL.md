# Marketplace Content Reviewer

## Purpose

Review marketplace content across title, attributes, description, rich content, images, video, reviews, questions, proof, and offer clarity.

## When to Use

Use this skill when the task is to review a marketplace card, find content gaps, prepare a relaunch, or improve conversion before ads and promotions.

## Inputs

Useful inputs: marketplace, product, category, current listing, images, video, reviews, questions, competitor cards, attributes, search terms, and sales signals.

## Workflow

1. Review all listing content elements.
2. Identify clarity, trust, proof, search, and objection gaps.
3. Compare content against buyer questions and competitors.
4. Prioritize fixes by conversion and search impact.
5. Recommend a clean content improvement roadmap.

## Output

Always provide content diagnosis, gap list, missing assets, proof gaps, search notes, prioritized fixes, and next steps.

## Common Mistakes

- Reviewing only text.
- Ignoring image and question gaps.
- Running ads before content is ready.
- Not using reviews as content input.
