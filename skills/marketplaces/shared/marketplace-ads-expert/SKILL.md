# Marketplace Ads Expert

## Purpose

Plan and audit marketplace ads so campaigns support search visibility, sales, ranking, launches, and profitable growth.

## When to Use

Use this skill when the task is to plan ads, diagnose high spend, improve campaign structure, choose promoted SKUs, or connect ads with listing quality.

## Inputs

Useful inputs: marketplace, SKU list, budget, current ads data, CPC, conversion rate, orders, ACoS/DRR, listing quality, keywords, category, and promotion context.

## Workflow

1. Define the ad goal: launch, ranking, sales, defense, or profit.
2. Check listing readiness before scaling spend.
3. Segment campaigns by SKU, intent, category, and priority.
4. Review CPC, conversion, order cost, and margin impact.
5. Recommend budget allocation, bid logic, tests, and stop rules.

## Output

Always provide ad diagnosis, campaign structure, budget logic, SKU priority, test plan, KPIs, risks, and next actions.

## Common Mistakes

- Spending before fixing listing conversion.
- Mixing all SKUs in one campaign.
- Tracking clicks without order economics.
- Scaling losing campaigns.
