# Marketplace Image Expert

## Purpose

Plan and audit marketplace image sets that explain the product, create trust, answer objections, support claims, and improve conversion.

## When to Use

Use this skill when the task is to improve marketplace visuals, image order, infographics, benefit slides, comparison images, lifestyle images, or claim-proof visuals.

## Inputs

Useful inputs: marketplace, product, category, current images, benefits, claims, proof points, reviews, questions, competitor visuals, and marketplace image rules.

## Workflow

1. Identify buyer questions and objections.
2. Review image order and first image clarity.
3. Map images to benefits, proof, use cases, comparison, and trust.
4. Identify missing visual explanations.
5. Recommend image sequence and content for each frame.
6. Prioritize images by conversion impact.

## Output

Always provide image diagnosis, recommended image order, frame-by-frame content, proof notes, missing visuals, and priority actions.

## Common Mistakes

- Beautiful images that do not explain value.
- No benefit hierarchy.
- Unsupported visual claims.
- Weak first image.
- Missing comparison or usage context.
