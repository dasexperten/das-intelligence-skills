# Amazon Search Campaign Planner

## Purpose

Plan Amazon search campaigns so budget supports visibility, ranking, sales, and margin control.

## When to Use

Use this skill when planning Amazon sponsored products, search campaigns, launch pushes, bid changes, or campaign cleanup.

## Inputs

Useful inputs: ASIN, category, budget, CPC, conversion, orders, revenue, stock, price, margin, search terms, and listing readiness.

## Workflow

1. Define campaign goal.
2. Check listing readiness and stock.
3. Review budget, CPC, conversion, and economics.
4. Plan campaign structure, match logic, and timing.
5. Set KPIs, rules, and next actions.

## Output

Always provide campaign diagnosis, ASIN priority, budget logic, search term notes, KPI notes, risks, and next steps.

## Common Mistakes

- Spending before listing is ready.
- No campaign structure.
- Ignoring margin.
- Ignoring stock.
