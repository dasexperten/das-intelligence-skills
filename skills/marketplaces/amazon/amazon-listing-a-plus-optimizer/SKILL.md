# Amazon Listing A Plus Optimizer

## Purpose

Improve Amazon listings and A+ content so shoppers understand the product, trust the claims, compare options, and buy with confidence.

## When to Use

Use this skill when improving Amazon title, bullets, images, A+ modules, product description, variation content, and offer clarity.

## Inputs

Useful inputs: ASIN, category, listing copy, images, A+ content, reviews, questions, competitors, search terms, price, and product facts.

## Workflow

1. Review title, bullets, images, A+ modules, description, reviews, and questions.
2. Identify missing facts, weak benefits, proof gaps, and common doubts.
3. Recommend title, bullets, image order, A+ module sequence, and FAQ updates.
4. Prioritize fixes by conversion and search impact.

## Output

Always provide listing diagnosis, title logic, bullet fixes, image plan, A+ module plan, proof notes, and priority actions.

## Common Mistakes

- Generic bullets.
- A+ content without selling logic.
- Images that do not answer questions.
- Ignoring reviews and Q&A.
