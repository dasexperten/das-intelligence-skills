# TikTok Shop Live Commerce Planner

## Purpose

Plan TikTok Shop live commerce sessions that explain products, answer doubts, create urgency, and connect viewers to purchase actions.

## When to Use

Use this skill when planning live selling sessions, live scripts, product demos, offer timing, host flow, or live performance review.

## Inputs

Useful inputs: product, host, audience, live goal, offer, product facts, demo steps, questions, schedule, and KPIs.

## Workflow

1. Define live goal and audience.
2. Build session flow: hook, demo, proof, offer, Q&A, repeat CTA.
3. Prepare product talking points and common answers.
4. Plan timing, assets, and KPI review.

## Output

Always provide live plan, run of show, product talk track, offer timing, Q&A notes, KPI notes, and next steps.

## Common Mistakes

- No clear session flow.
- Too much talking before product value.
- No prepared answers.
- No repeated CTA.
